tests.conftest module¶

This module setup the testing environment and make available the pytest fixtures

class tests.conftest.SparkETLTests(config_file: str = 'configs/config.json', extra_file_options: Dict = {})[source]¶

Bases: unittest.case.TestCase

This class does the testing setup for the spark code. Object of this class is available throughout testing session. Prepares gold standard data as mapped in the test_bed.json file Create necessary dataframes and tables, apply any schema defined in test_schema.py. Reads the job configuration defined config_file and append any testing specific or command line configurations if specified in the test_bed.json file. Defines mock objects to introspect the calling mechanism of a few spark libraries. Defines helper function to comapare the gold standard data with processes output.

Parameters

  • config_file (str, optional) – Relative path of the config file, defaults to ‘configs/config.json’

  • extra_file_options (Dict, optional) – Extra file options for the testing datafiles, defaults to {}

Variables

  • dataframes – Dictionary to hold dataframes for the testing, defaults to {}

  • mock_df – Mock object with spec of spark Dataframe to introspect the calling behaviour

  • mock_spark – Mock object with spec of SparkSession to introspect the calling behaviour
